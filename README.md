Subscription Data Analysis
📌 Project Overview

This project analyzes subscription-based customer data to understand revenue trends, customer behavior, and churn patterns. The goal is to help businesses improve retention and increase profitability using data-driven insights.

🎯 Objectives

Analyze active vs cancelled subscribers

Calculate Monthly Recurring Revenue (MRR)

Measure churn rate

Compare subscription plan performance

Identify revenue trends
